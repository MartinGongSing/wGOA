APS SOP
Version 1: 2014.11.11
  1. On Windows7 desktop, click AIM-APS icon
  2. File → New:
    Select folder: Look in: DATA/APS
    Files of type: APS 3321 Data file (*.A21)
    File name: yyyymmdd.A21
    Click Open
  3. Run → Auto Export...:
    Data types: Aerodynamic
    Units and Weights: Number / Raw Counts
    Delimiter: Comma
    Click OK
    Export as: confirm file name: yyyymmdd.txt
    Click Save
  4. File → Properties:
    Communications:
    • Select Com port 6
    Scheduling:
    • Sample length (sec): 297
    • Number of samples: 1
    • Total sample time: 4m 57s
    • Scheduled: Repeat every: 0h 5m
    • Start at time: any multiple of 5
    Click OK
