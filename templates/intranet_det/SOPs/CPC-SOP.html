CPC SOP
Version 2: 2015.05.25
Start measuring
  1. On Windows7 desktop, click AIM-CPC icon
  2. File → New:
    Select folder: Look in: DATA/CPC
    Files of type: CPC 3022 Data file (*.C22)
    File name: yyyymmdd.C22
    Click Open
  3. Run → Auto Export...:
    Extension: (*.csv)
    Delimiter: Comma
    Orientation: Column
    Output file name: yyyymmdd.csv
    Data point time format: Time Stamp <hh:mm:ss>
    Click OK
  4. File → Properties:
    Communications:
    • Select Com port 5
    Scheduling:
    • Sample length (sec): 86100
    • Number of samples: 1
    • Total sample time: 23h 55m
    • Averaging intervl (Sec): 300
    • Scheduled: Repeat every: 24h 0m
    • Start at time: any multiple of 5
    Click OK
Drain CPC
  1. On AIM-CPC, stop data acquisition
  2. On CPC's front panel, turn off the CPC pump: press PUMP
  3. On CPC's back panel, connect the fitting from the drain bottle to the LIQUID DRAIN connector
  4. Place the bottle below the CPC's base
    • help the liquid flow by removing all air from the drain tube
    • remove all liquid from the CPC's bottle to the "old butanol" bottle
  5. On CPC's front panel, press DRAIN/FILL button
    • the display reads Draining
  6. Disconnect bottle from CPC
Fill CPC (after draining)
  1. Fill bottle with new butanol to at least one-third full
  2. On CPC's back panel, connect the liquid-filled bottle to the LIQUID SUPPLY connector
  3. On CPC's front panel, press DRAIN/FILL button (only if CPC was previously in draining mode)
  4. On CPC's front panel, turn on the CPC pump: press PUMP
    • the display reads Filling or the actual reading, depending on configuration
      (CPC can be configured so that the liquid reservoir must be full to get a readout of the concentration)
